####### User Segments
1.ครู / ผู้ดูแลเด็ก (Primary User):
-เช็คช็คชื่อเด็ก
-บันทึกข้อมูลสุขภาพและพัฒนาการ
-อัปโหลดรูปกิจกรรมและภาพตรวจสอบ
-ทำแบบประเมินศูนย์
2.ผู้บริหารศูนย์ :
ดู Dashboard KPI
-ดูอันดับ Ranking ศูนย์
-ตรวจสอบปัญหาและข้อจำกัด
3.หน่วยงานกำกับดูแลระดับเขต/จังหวัด/ส่วนกลาง:
-ตรวจสอบคุณภาพศูนย์
-ดูสถิติรวมระดับประเทศ
-ใช้ข้อมูลในการปรับนโยบาย

####### ขอบเขตข้อมูลเด็กที่ต้องบันทึก
-ข้อมูลสุขภาพพื้นฐาน
    --น้ำหนัก
    --ส่วนสูงสู
    --BMI
    --ภาพรวมโภชนาการ (ผอม / สมส่วน/ อ้วน)
    --อาการเจ็บป่วย
    --ข้อจำกัดทางสุขภาพ
-พัฒนาการด้านต่าง ๆ
    --พัฒนาการทางร่างกาย
    --พัฒนาการทางอารมณ์
    --พฤติกรรม
    --การเรียนรู้และทักษะพื้นฐาน
-Tag ประกอบ
    --ข้อมูลการเบิกนม
    --อาหารที่บริโภค
    --ปริมาณอาหาร
    --ความถี่และคุณภาพสารอาหาร
    --บันทึกภาพกิจกรรมรายวัน

####### KPI ศูนย์เรียนรู้เด็ก
-ด้านความสะอาด
    1.ความสะอาดอาคาร
    2.อุปกรณ์เด็ก
    3.ห้องน้ำ
    4.พื้นที่อาหาร
    5.อัปโหลดรูปเพื่อยืนยัน
-ด้านการดูแลเด็ก
    1.สภาพร่างการเด็ก
    2.ภาวะโภชนาการ
    3.การเข้าเรียนสม่ำเสมอ
    4.ความปลอดภัยในศูนย์
-ด้านเอกสารและการจัดการ
    1.ความครบถ้วนในการบันทึกข้อมูล
    2.การส่งข้อมูลภายในเวลาที่กำหนด
-ด้านนวัตกรรมการเรียนการสอน
    1.การจัดกิจกรรมการเรียนรู้
    2.ผลงาน/หลักฐานกิจกรรม
